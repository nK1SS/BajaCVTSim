# BajaCVTSim
MATLAB Simulink simulation of a mechanical CVT (continuously variable transmission) for tuning applications.
This simulation is designed to simulate a Baja SAE vehicle from idle to acceleration.

RUN: Download all files to desired file location. Open "BajaCVTSim_RUN.m" to begin simulation. Once the UI opens, set the desired variables for tuning and run the simulation. 

MATLAB 2025a
