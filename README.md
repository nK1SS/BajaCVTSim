# BajaCVTSim
MATLAB Simulink simulation of a mechanical CVT (continuously variable transmission) for tuning applications.
This simulation is designed to simulate a Baja SAE vehicle from idle to acceleration. 

RUN: Download all files to desired file location. Open "BajaCVTSim_RUN.m" to begin simulation. 
Once the UI opens, set the desired variables for tuning and run the simulation. 

Engine preformance is based on the Kohler CH440 Restriced engine based on 2026 Baja SAE rules. 
Vehicle data is based on Tennessee Tech University Baja SAE vehicle.
CVT data is based on the GX9 Gaged mechanical CVT.

Future Edits: In the case of changes made to a the vehicle, engine, or CVT, The excel spreadsheet contains all data and is editable. 
The simulation automatically reads the excel file once the UI first opens.
